<Banner variant="normal" className="relative">
  ## O que é o Zeile Notebook?
</Banner>

Basicamente, é uma ferramenta de escrita de texto, com alguns fru-frus a mais.

É inspirada no Jupyter Notebook (que permite escrever código e texto).

<Banner variant="normal" className="relative">
  ## Limitações atuais
</Banner>

1. [ ] O compartilhamento de páginas ainda não está totalmente pronto.
2. [X] As únicas funções relacionadas ao usuário são as de cadastro, login e atualização de nome.
3. [ ] Alguns botões podem te levar pra uma página que ainda não existe, ou para uma página de onde você não consegue voltar ;) .
4. [X] Alguns pontos de autenticação ainda serão melhorados/corrigidos.
5. [X] Ainda não implementei login com o GitHub :P .
6. [X] Ainda não é possível criar tabelas (nem com markdown, nem com componentes de UI).
7. [X] Os metadados do componente de UI Callout não estão sendo salvos.
8. [X] Os componentes de UI Banner, Card e repositório do GitHub não são adequados pro tema claro.
9. [X] A edição de markdown fica com aparência estranha no tema claro.
10. [X] Todos os componentes de edição de texto ficam estranhos no tema claro.
11. [X] Ainda não é possível copiar páginas que foram compartilhadas.
12. [X] O componente de link para repositório do GitHub não está responsivo.
13. [X] O título na página inicial não está centralizado.
14. [X] Ainda não é possível adicionar notas de rodapé [^1].
15. [X] Ainda não é possível adicionar lista de tarefa.
16. [X] Ainda não é possível adicionar links diretamente.

[^1]: Essa é uma nota de rodapé!

<Banner variant="normal" className="relative">
  ## O que funciona no momento
</Banner>

1. Cadastro e login de usuário
2. Sincronização automática das páginas (salvando a cada 10s)
3. Renomear páginas.
4. Executar código de outras páginas, desde que estejam na mesma aba do navegador e que todos os blocos relacionados tenham sido compilados.
5. Compartilhar uma página, desde que ela esteja pública (dá pra editar a configuração no topo da página).
6. Reordenar blocos.
7. Utilizar componentes de UI (callout, card, banner e link para repositório do github).

## React/TypeScript

Você pode editar código abaixo.
Ele poderia ser importado em outro bloco, por meio de um import:
`import RelogioDigital from './Componente_13'`.

Isso não foi feito nesse exemplo por questões de renderização da página.

<TsxEditorMDX
  title="Componente_13"
  initialCode={`import React, { useState, useEffect } from 'react';\n\n\nconst estiloHora: React.CSSProperties = {\n  fontSize: '2rem',\n  fontWeight: 'bold',\n  color: '#169e69'\n};\n\nconst estiloContainer: React.CSSProperties = {\n  padding: '20px',\n  textAlign: 'center',\n  fontFamily: 'sans-serif'\n};\n\nconst RelogioDigital: React.FC = () => {\n  const [hora, setHora] = useState(new Date());\n\n  useEffect(() => {\n    const timer = setInterval(() => {\n      setHora(new Date());\n    }, 1000);\n\n    return () => clearInterval(timer);\n  }, []);\n\n  const horaFormatada = hora.toLocaleTimeString('pt-BR');\n\n  return (\n    <div style={estiloContainer}>\n      <h2>Hora Atual</h2>\n      <p style={estiloHora}>{horaFormatada}</p>\n    </div>\n  );\n};\n\nexport default RelogioDigital;`}
/>
